## Welcome to CapsuleX

This is your first blog post. You can edit it in `src/content/blog/hello-world.md`.

### Getting Started

To create a new post, add a `.md` or `.mdx` file to `src/content/blog/`:

```markdown
---
title: "Your Post Title"
description: "A brief description"
pubDate: 2026-07-04
tags: ["tag1", "tag2"]
---

Your content here...
```

### Features

- **Dark/Light Mode**: Automatically follows your system preference
- **Capsule Navigation**: Floating, frosted glass design
- **Fast**: Built with Astro for optimal performance

Enjoy writing! 🚀
