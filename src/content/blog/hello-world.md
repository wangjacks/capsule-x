## Welcome to CapsuleX

This is your first blog post. If you're seeing this, congratulations — you've taken your first step with this theme!

You can edit it in `src/content/blog/hello-world.md`.

### Getting Started

To create a new post, add a `.md` or `.mdx` file to `src/content/blog/`:

```markdown
---
title: "Your Post Title"
description: "A brief description"
pubDate: 2026-07-04
tags: ["tag1", "tag2"]
---

Your content begins here...
```

### Features

- **Dark/Light Mode**: Automatically follows your system preference. To be frank, this is nothing amazing.
- **Capsule Navigation**: Floating, frosted glass design. This is where the theme got its name.
- **Fast**: Built with Astro for optimal performance. I'm passionate about Astro, for real.

> This is my first time creating an Astro theme. Powered by AI, but the capsule is my idea. I'll keep refining this project for better performance. Hope you enjoy — let's begin the journey!
